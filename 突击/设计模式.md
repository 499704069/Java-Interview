# 一、单例

一个类只有一个实例

```java
package test;

/**
 * @Author: 98050
 * @Time: 2019-03-01 17:31
 * @Feature:
 */
public class Single {

    private static volatile Single single;

    private Single() {
    }

    public Single getSingleInstance(){
        if (single == null){
            synchronized (Single.class){
                if (single == null){
                    single = new Single();
                }
            }
        }
        return single;
    }
}
```
